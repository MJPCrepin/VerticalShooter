VertShooterDemo
====
Manuel Crepin (manuel@crep.in), Oct'17

## Current features
* Player object lifts off with any key
* Camera follows player max height
* Score based on peak height
* Floor deleted after liftoff
* Enemies spawn continuously on platforms
